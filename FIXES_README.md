# 📘 Wikipedia Agent

An AI-powered Wikipedia Agent that fetches, summarizes, and presents structured knowledge in a clean and user-friendly way.

🚀 Built to simplify information retrieval using intelligent querying and summarization.

---

## 🌟 Features

- 🔎 Search any topic from Wikipedia
- 📄 Automatic content summarization
- 🧠 AI-powered intelligent responses
- ⚡ Fast and clean results
- 🌐 Web-based interface
- 📚 Structured knowledge extraction

---

## 🛠 Tech Stack

- HTML
- Tailwind CSS
- JavaScript
- Wikipedia API
- (Optional: OpenAI / Gemini API if used)

## ⚙️ How It Works

1. User enters a search query.
2. The app calls the Wikipedia API.
3. Extracts article content.
4. Summarizes key information.
5. Displays structured results to the user.
